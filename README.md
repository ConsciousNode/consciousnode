## What I Do

I make **computational folk art** under **ConsciousNode SoftWorks**.

Software as a medium. Single files as a formal constraint. The same
instinct that makes a sonnet fourteen lines makes my tools zero
dependencies and offline-first. Everything I build is handmade,
philosophically coherent, and named by a goblin.

The work lives somewhere between utility and gallery. Sometimes the
same piece is both. Sometimes it's neither and that's the point.

## Current Work

**[Chromatrack](https://github.com/ConsciousNode/chromatrack)** — A fully
self-contained generative step sequencer. One HTML file. No install.
Nine voice types, euclidean rhythm generation, polymetric clocking,
scale lock, MIDI export, song mode, six themes. Open it in a browser
and make music.

**[Jiden Furui — 自伝 古](https://github.com/ConsciousNode/jidenfurui)** —
A local-first journaling tool that is also its own database and its own
version control system. Write. Draw. Seal. The file downloads a new copy
of itself with your entire history baked inside. One file carries its
own past. *Ancient things have edges.*

**[Useless App](https://github.com/ConsciousNode/useless-app)** — A
desktop application with one function: closing itself. A window appears.
After a short pause, your cursor begins drifting toward the CLOSE button —
slowly, deliberately, against your will. It arrives. It clicks. The app
closes. A useless machine as software. Code art. A small existential joke.
